# docsify-pytutor-plugin Demo

这是一个演示页。

## Python 示例

```pytutor-python
x = 1
y = 2
z = x + y
print(z)
```

## Python 简写示例

```pytutor
nums = [1, 2, 3]
for n in nums:
    print(n)
```

## Java 示例

```pytutor-java
public class Main {
    public static void main(String[] args) {
        int a = 3;
        int b = 4;
        int c = a + b;
        System.out.println(c);
    }
}
```
